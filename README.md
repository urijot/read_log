# Book Log App (read_log)

[English]
A simple book management application built with Next.js and Supabase.
- **Key Features**: Auto-fetching book covers via Google Books API, CRUD operations for your library.
- **Tech Stack**: Next.js, Supabase, Tailwind CSS, Vercel.

---

[日本語]
Next.js と Supabase を使用したシンプルな読書管理アプリです。
- **主な機能**: Google Books API を利用した表紙画像の自動取得、本の登録・一覧・削除。
- **使用技術**: Next.js, Supabase, Tailwind CSS, Vercel.
